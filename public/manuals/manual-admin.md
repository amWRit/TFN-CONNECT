This guide provides detailed instructions for managing the platform as an admin.

## Features & How-To

### 1. Admin Dashboard
**Access and overview:**
- Toggle "Admin Mode" from your user menu to access the admin dashboard.
- View live stats on users, jobs, opportunities, and posts.
- Use dashboard links to quickly jump to management sections.

### 2. User Management
**Manage the community:**
- Go to the user management section to view all registered users.
- Search, filter, and sort users by name, cohort, or status.
- Approve new users, edit user details, or deactivate accounts as needed.
- Reset passwords or update user roles (e.g., promote to staff/admin).

### 3. Content Moderation
**Keep the platform safe:**
- Review new posts, jobs, and opportunities for appropriateness.
- Remove or edit content that violates guidelines.
- Respond to user reports and take action as necessary.

### 4. Jobs & Opportunities
**Curate and manage listings:**
- Create new job postings and opportunities directly from the admin panel.
- Edit or remove existing listings.
- Highlight featured roles or programs to boost visibility.

### 5. Events
**Organize and manage events:**
- Add new events with details, agenda, and location.
- Edit or remove events as needed.
- Manage RSVPs and see attendee lists.

### 6. Analytics
**Track engagement:**
- Access analytics dashboards for user activity, posts, and engagement trends.
- Export reports for further analysis if needed.

### 7. Settings
**Configure the platform:**
- Update global settings, such as notification preferences and platform info.
- Manage system-wide announcements or banners.
